# Cryptography Challenges

This directory contains write-ups for cryptography challenges from SPYIREX CTF.

## Challenges

1. [Caesar Cipher](caesar-cipher.md) - Easy (75 points)
2. [Base64 Layers](base64-layers.md) - Easy (100 points)
3. [RSA Beginner](rsa-beginner.md) - Medium (300 points)

## Common Techniques

- Classical ciphers (Caesar, Vigenère, etc.)
- Base64/Base32 encoding
- XOR encryption
- RSA
- AES
- Hash functions (MD5, SHA, etc.)
- Frequency analysis

## Useful Tools

- CyberChef
- dcode.fr
- RsaCtfTool
- hashcat
- John the Ripper
- Python cryptography libraries (pycryptodome)
