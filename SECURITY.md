# Security Policy

## Scope

This repository primarily contains CTF writeups and reference material. It does
not host production services.

## Reporting A Security Concern

If you discover a security concern related to repository files, links, or shared
artifacts:

1. Do not publish sensitive details immediately.
2. Open a private report to maintainers first.
3. Include reproduction steps and affected paths.

## Response Expectations

Maintainers will review valid reports and respond as soon as possible.

## Safe Disclosure

After a fix is confirmed, responsible public disclosure is welcome.
