# Reverse Engineering Challenges

This directory contains write-ups for reverse engineering challenges from SPYIREX CTF.

## Challenges

1. [Simple Crackme](simple-crackme.md) - Easy (150 points)
2. [Anti-Debug](anti-debug.md) - Medium (300 points)
3. [Obfuscated Binary](obfuscated-binary.md) - Hard (400 points)

## Common Techniques

- Static analysis
- Dynamic analysis (debugging)
- Decompilation
- Anti-debugging bypass
- Code obfuscation
- Patching binaries
- String analysis

## Useful Tools

- Ghidra
- IDA Pro
- radare2/Cutter
- Binary Ninja
- gdb/pwndbg
- objdump, nm, readelf
- strings, file
- ltrace, strace
- x64dbg/x32dbg (Windows)
