# Code Of Conduct

## Our Commitment

This project is dedicated to a respectful, professional learning environment.
Contributors, participants, and maintainers are expected to collaborate in good
faith.

## Expected Behavior

- Be respectful in discussions and reviews.
- Focus on technical feedback and clear communication.
- Credit original challenge authors and sources.
- Keep submissions relevant to the repository scope.

## Unacceptable Behavior

- Harassment, personal attacks, or discrimination.
- Spam, trolling, or intentionally disruptive behavior.
- Posting private information without permission.

## Enforcement

Maintainers may edit, remove, or reject contributions that violate this code of
conduct.

## Reporting

Open an issue with a clear summary, or contact the maintainers privately if the
report should not be public.
