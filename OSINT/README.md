# OSINT Challenges

This directory contains write-ups for OSINT (Open Source Intelligence) challenges from SPYIREX CTF.

## Challenges

1. [Social Media Hunt](social-media-hunt.md) - Easy (100 points)
2. [Geolocation](geolocation.md) - Medium (250 points)
3. [Digital Footprint](digital-footprint.md) - Medium (300 points)

## Common Techniques

- Username enumeration
- Google dorking
- Reverse image search
- Geolocation from images
- Domain/IP investigation
- Social media analysis
- Metadata extraction
- Wayback Machine investigation

## Useful Tools

- Sherlock (username search)
- Maltego
- theHarvester
- SpiderFoot
- Google (advanced search operators)
- TinEye, Google Reverse Image Search
- ExifTool (metadata)
- Wayback Machine
- Shodan
- WHOIS lookup
- Creepy (geolocation)
