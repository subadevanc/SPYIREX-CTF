# SPYIREX-CTF

SPYIREX CTF (FLAG OVER FLOW) is a Capture The Flag event organized by the
Department of Cyber Security and Club CYBER NEXUS, Jerusalem College of
Engineering.

This repository contains challenge descriptions and post-event writeups.
All flags follow the format `JCE{...}`.

## Event Snapshot

- Event date: 24 January 2026
- Challenge authors: `0xr0h4n`, `Subadevan C`, `orochi`, `N1rm4l_D4rkn3ss18`, `d3xt3r`
- Categories used in event: Cryptography, Web, Reverse Engineering, Forensics, OSINT, Steganography

## Repository Structure

- `Reverse Engineering/` - APK and binary reversing writeups
- `Web/` - Web and WASM-based challenge writeups
- `Steganography/` - File/media hiding challenge writeups
- `Forensics/` - Forensics challenge index
- `imports/pdf_extract/` - Imported source PDF, extracted text, and images
